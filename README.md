# DJ SOUL - Presskit Profesional 🎛️

Presskit interactivo y profesional para **DJ SOUL**, diseñado para destacar su trayectoria, setup técnico y servicios en Santiago de Chile.

🌍 **Live Demo:** [djsoul.tusitioya.cl](https://djsoul.tusitioya.cl/)

## 🚀 Características del Proyecto
- **Diseño Urbano & Asimétrico:** Paleta de colores vibrantes (amarillo neón, rojo intenso, negro asfalto).
- **Fondo 3D Inmersivo (Three.js):** Túnel espacial con partículas interactivas que reaccionan al movimiento del ratón.
- **Efectos CSS Personalizados:** Animaciones al ritmo del reggaeton (~95 BPM) en el logotipo, simulando la vibración de un parlante.
- **CSS Art:** Ilustraciones 100% hechas en código puro (CSS) para representar el hardware técnico (Pioneer DDJ-FLX6, Launchpad X, Laptop Asus, Rig de Streaming).
- **Responsive Design:** Adaptado perfectamente a dispositivos móviles y de escritorio.
- **SEO & Redes Sociales:** Open Graph Meta Tags optimizados para compartir en WhatsApp, Instagram y Facebook con el logotipo oficial.

## 🛠️ Tecnologías Utilizadas
- **HTML5** semántico
- **CSS3 Vanilla** (Keyframes, CSS Grid, Flexbox, Pseudo-elementos)
- **Vanilla JavaScript**
- **Three.js** (WebGL para animación 3D del fondo)
- **GitHub Pages** (Hosting)
- **Cloudflare** (DNS y CNAME)

## 📁 Estructura del Proyecto
- `index.html`: Archivo principal con toda la estructura, CSS embedido y lógica 3D.
- `logo.jpg`: Logotipo oficial utilizado para animaciones y metadatos de redes sociales.
- `CNAME`: Configuración de dominio personalizado.
